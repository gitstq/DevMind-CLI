# 📝 更新日志

所有重要的项目更新都会记录在此文件中。

## [1.0.0] - 2026-06-01

### 🎉 首发版本

#### ✨ 新增功能

- **代码分析器** (`analyze`)
  - 多语言代码结构分析
  - 代码统计（文件数、行数、语言分布）
  - 复杂度评估
  
- **代码审查器** (`review`)
  - 安全漏洞检测
  - 性能问题识别
  - 代码风格检查
  
- **重构顾问** (`refactor`)
  - 智能重构方案
  - 问题代码定位
  - 重构前后对比
  
- **文档生成器** (`docgen`)
  - 自动生成 docstring
  - Markdown 文档生成
  - 多格式支持（markdown/html/rst）
  
- **代码扫描器** (`scan`)
  - 安全扫描
  - 质量扫描
  - 风格扫描
  
- **交互式 TUI** (`chat`)
  - 自然语言对话
  - 代码片段分析
  - 命令执行
  
- **项目记忆系统**
  - 项目上下文学习
  - 模式记忆
  - 洞察记录

#### 🔧 技术特性

- ✅ **零依赖设计** - 仅使用 Python 标准库
- ✅ **GLM-5.1 驱动** - 智谱AI长程任务模型
- ✅ **多语言支持** - 简体中文、繁体中文、English
- ✅ **MIT 协议** - 完全开源

#### 📦 支持的语言

- Python
- JavaScript / TypeScript
- Java
- C / C++
- Go
- Rust
- Ruby
- PHP
